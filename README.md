# Shift Rotation Engine

Base técnica del proyecto para el sistema de rotación de turnos.

## Requisitos

- Node.js 20 o superior
- npm 10 o superior

## Instalación

```bash
npm install
```

## Scripts disponibles

```bash
npm run dev
npm run test
npm run lint
npm run typecheck
npm run build
npm run format
```

## Estructura

- `src/application`: casos de uso en fases futuras.
- `src/pipeline`: orquestación en fases futuras.
- `src/domain`: dominio del negocio en fases futuras.
- `src/infrastructure`: adaptadores externos en fases futuras.
- `src/shared`: tipos y utilidades técnicas compartidas.
- `tests`: pruebas unitarias e integración.
- `docs`: documentación técnica incremental.

## Estado actual

Este bootstrap no contiene lógica de negocio.
Solo prepara el proyecto para comenzar la implementación del dominio.
